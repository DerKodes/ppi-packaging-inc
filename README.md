# PPI Packaging Inc. Website 📦

> The official front-end website for PPI Packaging Inc., showcasing products, services, and company information with a modern, responsive design.

**Live Website:** [ppipackaginginc.com](http://ppipackaginginc.com)

## 📖 Project Overview

This repository contains the production code for the PPI Packaging Inc. website. The primary goal of this project was to establish a professional digital presence for the company, ensuring a seamless user experience across all devices. The site features optimized asset loading, intuitive navigation, and interactive UI components.

## 💻 Tech Stack

This project was built utilizing the following technologies:

* **Markup & Style:** HTML5, CSS3
* **Scripting:** JavaScript (ES6+)
* **Frameworks/Libraries:** * Bootstrap 5 *(for responsive layout and styling)*
    * SweetAlert *(for custom, elegant popup notifications and alerts)*

## ✨ Key Features

* **Fully Responsive Design:** Adapts flawlessly to desktop, tablet, and mobile screens.
* **Modern UI/UX:** Clean, accessible interface tailored to the packaging industry.
* **Interactive Elements:** Smooth scrolling, dynamic active states, and custom alert modals.
* **Performance Optimized:** Lightweight assets and structured code for fast loading speeds.
* **Cross-Browser Compatibility:** Tested and functional across all major web browsers (Chrome, Firefox, Safari, Edge).
